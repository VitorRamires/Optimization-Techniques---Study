 <h2><span>🚀</span> Projeto de Treino: Otimização e Performance em React</h2>
<h4>Este projeto foca no aprofundamento de técnicas de performance e no entendimento do funcionamento interno do React.</h4>

</br>

🧠 O que foi explorado:
<ol>
<li>Memoização: Aplicação prática de memo, useMemo e useCallback para evitar renderizações desnecessárias.</li>

<li>Reconciliation: Estudo de como o React identifica e modifica o DOM.</li>

<li>Diffing Algorithm: A importância real das keys em listas e componentes.</li>

<li>Rendering: Entendimento de conceitos como State Batching para otimizar atualizações de estado.</li>
</ol>
